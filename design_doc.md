# NekoBIN（ねこビン）ブランドサイト兼LP 制作設計図

このドキュメントは、ペットとの思い出を美しく残すブランド「NekoBIN」の公式サイトおよびLPの制作計画です。

## 1. プロジェクト概要
「愛する毛を、思い出に変える。」をコンセプトに、ペットの毛を保管する「ペット愛毛ケース」という新しいカテゴリーを提案します。単なるメモリアルグッズではなく、元気な今から使い始める「小さなタイムカプセル」としての価値を伝えます。

- **サイト名**: NekoBIN (ねこビン) ブランドサイト
- **メインメッセージ**: 「愛する毛を、思い出に変える。」
- **カテゴリー定義**: ペット愛毛ケース (商標・新カテゴリー)
- **ゴール**: 新カテゴリーの認知、プレミアムレンジでのポジショニング確立、ファン化、将来的なプラットフォームへの入口。

## 2. 戦略的ポジショニング（カテゴリー戦略）
単なる「毛を保管する容器」という機能的価値から脱却し、独自の市場を創出します。

### カテゴリーの創出（ほりにしモデルの応用）
- **市場の再定義**: 競合は安価なカプセルトイや画一的なメモリアルグッズではなく、「高価な写真アルバム」「スマホ内のデジタルデータ」「家族の歴史を刻むタイムカプセル」といった「思い出を保存する手段」へと転換。
- **ベネフィット競合の転換**: 数千円の価格設定を、容器の価格ではなく「家族の絆を永遠に保存するための投資」として正当化。

### 4Cモデルに基づく定義
1. **Customer Problem**: 重すぎるメモリアルと、チープすぎるケースの間の「温かみのある中価格帯〜プレミアム層」の欠如。
2. **Category Value**: 日常から旅立ちの日まで、ペットとの時間を蓄積する「フィジカル・アンカー」。
3. **Category Keyword**: **「ペット愛毛ケース」**（遺毛という言葉を避け、生気と愛情を感じさせる造語）。
4. **Category Perception**: お迎え記念日や誕生日のスタンダードなギフトとしての文化的定着。

## 3. ターゲット
- ペット（猫、犬、うさぎ等）を家族として大切にしている人。
- ブラッシング後の毛を捨てることに寂しさを感じている人。
- ペットが健康なうちから、形に残る思い出を作りたい人。
- 動物福祉（保護猫・保護犬活動）に関心がある人。
- ペット好きの友人への特別なギフトを探している人。

## 3. デザインシステム
プレミアムで温かみがあり、情緒に訴えかける「シネマティック」な世界観を構築します。

### カラーパレット
- **Base**: オフホワイト (#F9F8F6)、生成り
- **Accent**: やわらかいベージュ、木目に合うブラウン (#5D4037)
- **Detail**: アンティークゴールド、真鍮色 (#B8860B)
- **Deep**: 深いネイビー (#1A237E)、オリーブグリーン (#556B2F)
- **Concept**: 自然光、木の温もり、手触り感を感じさせる配色。

### タイポグラフィ
- **見出し (Headings)**: 上品な明朝体 (例: "Shippori Mincho", "Noto Serif JP")
- **本文 (Body)**: 読みやすいゴシック体 (例: "Noto Sans JP")

### ビジュアル・動き
- **画像**: 自然光、木目、ペットの毛並み、飼い主の手、日常の風景。
- **アニメーション**: スクロール時のふわっとした表示（フェードイン）、ボタンの優雅なホバー、派手すぎない上品な遷移。

## 4. プレミアム・エクスペリエンス設計
ブランドの価値を最大化し、ユーザーの感性に深く訴えかけるための詳細設計です。

### ブランドの「声（トーン＆マナー）」
- **世界観**: 詩的で、温かく、静かな感動を呼ぶトーン。
- **言葉遣い**: 「〜です・ます」を基本とし、ユーザーに寄り添うような優しく、かつ品格のある表現。
- **NGワード**: 安っぽさを感じさせる言葉（激安、コスパ、格安など）は使用せず、「価値」「一生もの」「想い」を強調。

### 感性に訴えるマイクロインタラクション
- **静かな動き**: 唐突な動きを避け、すべての遷移を柔らかなフェードや滑らかなスライドで実装。
- **質感の反映**: 真鍮パーツに光が当たったような繊細な光沢エフェクトや、スクロールに連動した微細な視差効果（パララックス）。
- **タイポグラフィの演出**: 重要なコピーは、一文字ずつ呼吸するようにゆっくりと現れるアニメーションを採用。

### ビジュアルの「質感（テクスチャ）」へのこだわり
- **背景テクスチャ**: デジタルな平坦さを避け、わずかにリネンや上質な紙のような質感を背景に重ねる。
- **マテリアルの強調**: 真鍮の重み、ガラスの透明感、ペットの毛の柔らかさを強調した高精細なビジュアル。

### 信頼を醸成する「透明性」と「ストーリーの深掘り」
- **制作背景**: 商品の素材選定や、一つひとつに込められた職人（または開発者）の想いを詳細に記述。
- **社会的意義の明文化**: 寄付金の使途や支援先の情報を具体的に示し、「購入が善行につながる」納得感を提供。

### ギフト体験の視覚化
- **贈るシーンの提案**: 大切な人へ贈る際のパッケージング、リボン、メッセージカードの美しさを視覚的に紹介。
- **UX**: 「自分用」だけでなく「大切な人へ」という導線を明確に設計。

### SEOキーワード戦略
- **メインキーワード**: ペット愛毛ケース、遺毛 保管、猫 抜け毛 活用
- **サブキーワード**: ペット メモリアル ギフト、おしゃれ 抜け毛 ケース、ブラッシング 記念、保護猫 寄付
- **手法**: セマンティックHTML、適切なAltタグ、構造化データの検討。

## 5. サイト構成詳細（MVP構成）
初心者でも着実に制作できるよう、本質的な価値を伝える6つのセクションに絞り込みました。すべて1カラム（上から下への流れ）で構成し、レスポンシブ対応を容易にします。

1. **ファーストビュー (Hero)**
   - **内容**: メインコピー「愛する毛を、思い出に変える。」と象徴的なビジュアル。
   - **目的**: 訪問者の心を一瞬で掴み、世界観を提示する。
2. **ブランドストーリー (Story)**
   - **内容**: 「ペット愛毛ケース」という新カテゴリーの提案。ブラッシングの毛を捨てる寂しさを思い出に変える想い。
   - **目的**: 共感を生み、商品の必要性を理解してもらう。
3. **商品ラインナップ (Products)**
   - **内容**: ねこビン / いぬビン / うさビン。シンプルな画像、価格、購入ボタン。
   - **目的**: 検討・選択をスムーズにし、購入（外部サイト）へ誘導する。
4. **使い方 (How to Use)**
   - **内容**: 1.ブラッシング → 2.集める → 3.入れる → 4.飾る → 5.見返す の5ステップ。
   - **目的**: 使用シーンを具体化し、購入後の楽しみをイメージさせる。
6. **社会的意義 (Social Impact)**
   - **内容**: 寄付の理念、支援先団体の紹介、活動レポートの予定。
   - **目的**: 購買行動自体が動物たちの命を救う支援になる構造を提示し、プレミアム価格への納得感を提供。
7. **未来構想 (Our Vision)**
   - **内容**: ARによる思い出再生、AIによる動画生成構想の提示。
   - **目的**: 単なるメーカーではなく、記憶を軸としたプラットフォーマーとしての期待感を醸成。
8. **フッター (Footer)**
   - **内容**: SNSリンク、簡易的なお問い合わせ窓口。

## 6. 技術スタック
- **Frontend**: HTML5, Vanilla CSS, Vanilla JS（最小限）
- **Layout**: CSS Flexbox/Grid を活用したシンプルな1カラム。
- **Icons/Fonts**: Google Fonts ("Shippori Mincho", "Noto Sans JP")
- **SEO**: メタタグと適切な見出し構造。

## 7. セキュリティと安全性
初心者がサイトをインターネットに公開する際、トラブルに巻き込まれないための安全な設計指針です。

- **機密情報の完全排除 (No API Keys in Frontend)**: JavaScriptコード内にAPIキーなどを直接記述しません。
- **安全なリンク設定**: 外部サイトへのリンクには `rel="noopener noreferrer"` を付与します。
- **HTTPSの推奨**: 公開時はHTTPSが適用される環境（GitHub Pages等）を利用します。

## 8. 制作のポイント（初心者向け）
- **まずは動かさない**: アニメーションは最後。まずは文字と画像が正しく並ぶことを優先。
- **画像サイズに注意**: 適切なサイズにリサイズしてから使用。
- **お問合せはSNSで**: フォーム作成の難しさを避け、SNSのDMへ誘導。
